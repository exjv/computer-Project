# 第6步：系统门户业务首页重构

## 本步完成

1. 首页前端重做为业务门户（不是普通欢迎页）：
   - 系统介绍
   - 所属单位信息
   - 当前通知公告
   - 快速入口
   - 待办工单
   - 统计摘要
2. 公告不再做独立菜单入口，直接在首页展示。
3. 新增后端工作台聚合接口，按角色返回：
   - 快速入口
   - 待办摘要
   - 统计摘要
   - 公告信息
4. 快捷入口基于权限码进行前端可见性控制。
5. 首页布局优化，卡片化展示，强化业务感。

## 本步接口变化

- `GET /api/portal/workbench`：新增工作台聚合接口。
- `GET /api/portal/home`：继续用于登录门户公开信息。

## 本步新增页面

- 无新增路由页面。
- 重构 `HomeView` 作为业务首页。
